# CRUD Data Barang Laravel

## Deskripsi
Project ini adalah aplikasi CRUD sederhana menggunakan Laravel, PHP, dan MySQL untuk mengelola data barang.

## Fitur
- Tambah data barang
- Tampilkan data barang
- Update data barang
- Hapus data barang

## Cara Instalasi
1. Clone project ini
2. Masuk ke folder project
3. Jalankan: composer install
4. Copy .env.example menjadi .env
5. Atur database di file .env
6. Jalankan: php artisan key:generate
7. Jalankan: php artisan migrate
8. Jalankan: php artisan serve
9. Buka di browser: http://127.0.0.1:8000
